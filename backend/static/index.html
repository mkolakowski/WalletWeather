Tags

Tags are lighter than categories — each transaction has one category, but can carry any number of tags (like “vacation”, “reimbursable”, or “business”). Use them to slice spending across categories without reshuffling your budget.

Auto-tag rules

When a transaction's description contains the pattern (case-insensitive substring match), the chosen tag is auto-attached on save and on CSV import. Add an optional category filter to scope the rule. Use Backfill to apply every active rule to your existing transactions retroactively.

Permissions

Grant other users access to your accounts. Owner can manage all permissions including other owners, and can delete the account. Edit can modify data, run backups, and manage read/edit permissions, but cannot touch owners. Read can view the forecast only. Deny hides the account. A ★ next to an account name means you own it.
